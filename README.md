# Programming_Project2
Project 2

This project is an interactive Tic-Tac-Toe game on a 3x3 grid with:
A human player vs AI opponent
A build in tutor that can suggest the most optimal move
Follows standard rules of Tic-Tac Toe

project.py

Starts the game, how to play the game with a build in "tutor". Wures together the game logic.

## Requirements
- project.py
Has all the required contents of the game, games.py and utils.py is not needed to run the code.
- numpy
Needed for random number generation
- pip
To install numpy, use
```pip install numpy```
